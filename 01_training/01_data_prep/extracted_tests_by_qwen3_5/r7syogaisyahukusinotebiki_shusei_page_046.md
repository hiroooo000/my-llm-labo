

# 4 手当・年金・貸付等

## (1) 心身障害者等福祉手当（区制度）
**身知難児**

身体障害者手帳・愛の手帳をお持ちの方、難病等になり患し基準を満たしている方に支給される手当です。

### 対象・手当額

| 対象<br>（対象者の種別に2つ以上該当しても、<br>重複して支給を受けることはできません。） | 手当月額<br>（4・8・12月の原則20日に支給） |
| :--- | :---: |
| **①** 身体障害者手帳 1・2 級<br>愛の手帳 1・2・3 度<br>脳性麻痺<br>進行性筋萎縮症<br>※「脳性麻痺」と「進行性筋萎縮症」の方は、手帳の等級に関わらず対象となります。<br>※20歳未満の方は児童育成手当の受給対象となります。 | 15,500円 |
| **②** 身体障害者手帳 3 級<br>愛の手帳 4 度 | 13,500円 |
| **③** 指定の疾病（→201ページ）にかかり、医療券等（※）を所持している方 | 15,500円 |

※ 医療券等の種類
*   A 「難病の患者に対する医療券等に関する法律」の対象疾病による医療受給者証
*   B 東京都難病医療費等助成制度の医療券（→59ページ）
*   C 小児慢性特定疾病の医療費助成制度の医療券のうち（→59ページ）、A・Bと同種の疾病と認められるもの

### 受給できない方
1.  障害者手帳の交付を受けた年齢又は特定医療費（指定難病）受給者証の申請をした年齢が65歳以上の方
2.  児童育成手当（障害手当）の受給者
3.  施設に入所している方
4.  本人もしくは扶養義務者（対象者が20歳未満の場合）の所得が限度額を超える方（→204ページ）

### 申請手続
次のものをお持ちの上、窓口で申請してください。
1.  身体障害者手帳又は愛の手帳
2.  上記※に該当する医療受給者証
3.  預金通帳又はキャッシュカード（※本人名義）
4.  申請者のマイナンバーカード又は通知カード
