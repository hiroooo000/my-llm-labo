# （2）障害福祉サービス等の種類

## 【障害者総合支援法に基づくサービス】

| 3 | 種　類 | 内　容 |
| :--- | :--- | :--- |
| 障害者総合支援法及び看護福祉法のサービス | 居宅介護<br>（ホームヘルプ） | 居宅における入浴・排せつ等の介助、調理・掃除・洗濯等の援助、通院の介助を行います。 |
| | 重度訪問介護 | 重度の肢体不自由者又は重度の知的障がいもしくは精神障がいにより行動上、著しい困難を伴う障がい者に対し、移動の援助、外出時に必要な入浴・排せつ又は食事の介助、調理・掃除・洗濯等の家事やその他の生活援助、外出時における移動援助等を総合的に行います。 |
| | 同行援護 | 視覚障がいのある方に対し、外出時に必要な援助、視覚の補助を行います。 |
| | 行動援護 | 知的障がい又は精神障がいにより行動上著しい困難を伴う障がい者に対し介護を必要とする方に対し、行動上に必要な援助、外出時に必要な援助、その他の介助等を行う必要な援助を行います。 |
| | 介護等包括支援サービス | 介護の必要性がともうた方に対し、居宅介護その他の障害福祉サービスを包括的に行います。 |
| | 短期入所<br>（ショートステイ） | 短期入所により、夜間も含めて施設において、入浴・排せつ又は食事等の介護や日常生活の支援を行います。 |
| | 障害者総合支援法及び看護福祉法のサービス | 施設入所支援<br> | 施設入所の利用者に対して、夜間における入浴・排せつ又は食事等の介護等を行います。 |
| | 訓練等給付 | 自立訓練<br>（機能訓練・生活訓練） | 地域住民が生活社会ができるよう、主として障がい者において、定期的な身体機能又は生活能力の向上のための訓練を行います。 |
| | | 宿泊型自立訓練 | 地域移行に向けて、定期的に、障がい者の設備を利用して日常生活能力を向上させるための支援を行います。 |
| | | 就労移行支援 | 一般企業等での就労を希望する方に対し、定期的に、就労に必要な知識及び能力向上のため訓練を行います。 |
| | | 就労選択支援 | 就職・就業について、必要な選択が出来るよう、就労アセスメントの手法を活用して支援を行います。 |
| | | 就労継続支援<br>（A型・B型） | 一般企業等での就労が困難な方に対し、働くための知識及び能力向上のため訓練を行います。 |
