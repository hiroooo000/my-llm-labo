# 17 余暇活動の支援

## (1) 心身障害児・児レクリエーション　　　　　　　　　　　　　　　　　　　　　　身知的

障害のために、日帰行事の機会が少ない方を、年1回バス旅行にご招待します。

| | |
| :--- | :--- |
| 対象 | 区内在住の心身障害者又は肢の不自由をお持ちの方 |
| 実施方法 | 日帰りバス旅行 |
| 申込 | 旅行先・日程・申し込み方法等詳細は、区報「ぶんきょう」、区ホームページでお知らせします。 |

TEL 03-5803-1212　FAX 03-5803-1352

## (2) 障害者団体バス借上経費助成

区内の障害者・児及びその保護者等によって構成する団体が、バスを借り上げて事業を行った場合に、借上経費の一部を助成します。助成に当たるには、実施日の1か月前までにあらかじめ申請が必要です。

問い合わせ先　障害福祉課障害者在宅サービス係
TEL 03-5803-1212　FAX 03-5803-1352
