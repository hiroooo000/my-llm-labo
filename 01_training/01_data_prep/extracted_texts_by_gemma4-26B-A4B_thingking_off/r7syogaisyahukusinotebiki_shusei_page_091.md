将来の自立した生活に向けて養護と同園している精神病院から退院する予定の方が単身生活を体験することを希望されるとき、区が委託する社会福祉法人が宿泊場所を提供し、支援を行います。原則として事前登録が必要です。

# (16) 緊急時受け入れ支援事業

ご家族の疾病や障害などにより、障害者が一人になって生活できなくなるとき、または他のご家族では介助できないとき、ご本人を一時的に受け入れ日常生活の支援をします。
利用するには、事前の登録が必要です。登録を希望される場合は、支援を受けている障害者福祉サービス事業所や地域生活支援局（ $\rightarrow$ 24ページ）などにご相談ください。

| 対象 | 区内在住の18歳以上65歳未満の障害者 |
| :--- | :--- |
| 利用できるとき | ①ご家族の突然の疾病や不在になるとき<br>②障害者へ疾病が発生したときや障害のおそれがあるとき |
| 受付時間 | 午前7時～午後10時 |
| 利用料金 | 無料。ただし、食費やリネン費などの実費負担あり。 |
| 利用日数 | 3泊4日です。 |
| 電話番号 | 03-5981-9597（社会福祉法人武蔵福祉） |
| ファックス番号 | 03-5981-9598 |
