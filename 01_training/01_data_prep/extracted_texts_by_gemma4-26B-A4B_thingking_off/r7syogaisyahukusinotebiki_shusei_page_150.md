# 16 仕事

## (1) 文京区障害者就労支援センター

障害者の就労に関する相談窓口として、障害者や家族・企業・関係機関からの相談を受け、障害者施設やハローワークと連携しながら、各種就労に向けての支援を行います。

| 支援 | 支援の内容 |
| :--- | :--- |
| 就労準備支援 | 適性や能力に応じた就労への支援 |
| 就労準備セリング | キャリアカウンセリング、就職訓練の支援、就職活動支援、履歴書・職務経歴書の作成支援、模擬面接や面接同行等 |
| 職場実習支援 | 就労を目指す方へ、適職を見つけるための職場実習をコーディネート。実習先での事業主等に対する理解を求め、職場環境の調整を行う等の支援 |
| 就職継続支援 | 安心してお仕事を続けられるように、就職後の継続したフォロー |
| 生活支援 | 聞くことを楽しむために必要な、生活リズムの調整、健康管理、余暇などの職業生活に関する相談・助言 |
| 障害者雇用に取り組み企業等への支援 | 企業の障害者雇用の取り組みに関する助言、雇用の雇用管理やキャリア支援に関する助言、雇用における合理的配慮等に関する助言 |

| 費用 | 無料 |
| :--- | :--- |
| 受付 | 月〜金曜 午前9時〜午後5時30分<br>(祝日及び年末年始を除く) |

## (2) 波田須橋公民館職業安定所（ハローワーク波田須）

就職することを希望する障害者の方への仕事の情報提供、紹介、就職活動の相談、必要な技能習得のための職業訓練への入校相談を行っています。
また、障害者の募集、雇用する企業への紹介、関連する制度等の相談、情報提供を行っています。

| 受付 | 月〜金曜 午前8時30分〜午後5時9分 9階窓口 |
| :--- | :--- |

※手による必要な支援について、お気軽にご相談ください。
※就労相談は、登録制です。
※就職を希望する障害者の方を対象としています。相談日を設けています。相談手続きに関する日程については、お問い合わせください。
