# （７）ひとり親家庭医療費助成　　　　　　　　　　　　　身分証明

ひとり親家庭の保護者の対象の子どもに、保護除の自己負担の医療費助成を行っています（入院時食事療養標準負担額を除きます）。

| 対象 | 次の①～⑧のうち１つに該当する子ども（※要件あり） |
| :--- | :--- |
| | ※対象となる子どもの年齢<br>・18歳まで（18歳の誕生日後の3月31日まで）<br>・中途以上の障害がある場合は、20歳未満 |

| 5 医療 | 要件 | ①父母が離婚した<br>②父又は母が死亡<br>③父又は母が1年以上死不明<br>④父又は母は1年以上拘禁されている<br>⑤父又は母が暇からのDVで、裁判所からの保護命令を受けている<br>⑥父又は母が法令により1年以上拘禁されている<br>⑦婚姻により離別して生きて、父又は母から扶養されていない<br>⑧父又は母が重度の障害を有する |
| :--- | :--- |
| | 助成内容 | 対象の子どもと保護者の保険除に係る医療費自己負担のうち、<br>■部分は<br>・本人、扶養養族とも住民税課税の自己負担（3割）を全額助成<br>・本人は扶養養族とも住民税非課税の自己負担（3割）のうち3分の2助成 |

# （８）後期高齢者医療制度への加入　　　　　　　　　　　　身分証明

後期高齢者医療制度は、75歳以上の方を対象とした医療費ですが、65歳から74歳の方で一定の障害がある方は、任意で加入することができます。加入するためには、申請により定めることがあります（75歳以上の方は申請不要で自動切替）。

| 認定要件 | 65歳から74歳の下に該当する方<br>①障害年金（1～2級）受給者<br>②身体障害者手帳（1～4級）<br>※4級は肢体障害の一部と音声・語学機能障害に限る。<br>③愛の手帳（1、2度）<br>④精神障害者保健福祉手帳（1～2級）等 |
| :--- | :--- |
| 必要書類 | ①国民年金の年金証書、身体障害者手帳、愛の手帳、精神障害者保健福祉手帳等<br>②申請者のマイナンバーカード、届出番号が記載された住民票の写し等 |
