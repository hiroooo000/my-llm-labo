# （５）中等難聴発達事業補聴器購入助成

18歳未満で、身体障害者手帳の交付を受けることができる聴力ではない中等難聴の方に対して、補聴器の購入費用の一部を助成します。なお、助成を受けるには、事前の相談が必要になります。

| 対象 | 次のいずれに該当する方 |
| :--- | :--- |
| | ①区内在住 |
| | ②18歳未満の方 |
| | ③身体障害者手帳の交付を受けることができる聴力ではない方 |
| | ④両耳の聴力がおむぬ30dB以上で、補聴器の装用により言語の獲得等、一定の効果が期待できると医師が判断する方であること |

※令和6年度より、所得制限を撤廃しました。
