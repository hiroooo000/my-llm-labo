# 【育成医療】

身体に障害のある子に対し、生活力を得るために必要な医療の給付を行う制度です。次の①～④の条件を満たした子の医療費の一部を助成します。
① 18歳未満の子で機能障害があり、手術等により確実な治療効果が得られること
② 指定自治支援（育成）医療機関で療養が診察し、治療していること
③ 保護者が区内に住所を有していること
④ 区の住民税所得割額が25万円未満であること（一部例外あり）

| 対象障害 |
| :--- |
| ①肢体不自由 ②視覚障害 ③聴覚・平衡機能障害 ④言語・聴覚機能障害 ⑤認知機能障害 ⑥精神障害 ⑦小児癌 ⑧肝臓障害 ⑨その他の先天性内臓障害 ⑩免疫怠慢ウイルスによる免疫機能の障害<br>※歯科矯正（顎口蓋裂等に限り）ります。<br>※歯科矯正については、通院のみ対象です。<br>※けいへいニア、国のタク医療は対象外です。 |

| 給付内容 |
| :--- |
| 指定医療機関における対象障害の療養にかかる診療費の自己負担分について、一部助成します。 |
