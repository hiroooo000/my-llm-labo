## (12) 「四季の郷 温泉やまごこ荘」の利用　　　　　　　　　　身 寄 備 備

区内在住者が「四季の郷 温泉やまごこ荘」に宿泊するには助成があり、障がい者及び随伴者に、補助金額が増額されます。

| 対象 | 区内在住での①～④の手をお持ちの方 |
| :--- | :--- |
| | ① 身体障がい者手帳 |
| | ② 愛の手帳 |
| | ③ 精神障がい者保健福祉手帳 |
| | ④ 聴覚障 guitarist 手帳 |
| | ⑤ 視覚障がい者手帳 |
| | 随伴者：区内在住での①～④に該当する方に |
| | (随伴者は随伴を要する障がい者1名につき1名まで) |

| 金額 | 大人（中学生以上） 4,000円（利用料金9,500円のうち3,250円補助） |
| :--- | :--- |
| | 子ども（小学生） 3,150円（利用料金6,400円のうち3,250円補助） |
| | ＊障がい者及び随伴者1人当たりの1泊2食の料金（大人は入浴別途） |
| | ＊1泊利用は500円追加 |
| | ＊土曜日・休日前日は1,000円追加 |
| | ＊12月31日から1月3日まで宿泊の料金は2,000円追加 |
