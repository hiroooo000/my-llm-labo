# (14) 東京都育成庁地域支援部生涯学習課 事業

視覚・聴覚障害者のための講習等を行っています。費用は原則無料です。

| 講座名 | 対象 | 内容 |
| :--- | :--- | :--- |
| 視覚障礙者教養講座 | 都内の視覚障礙者及びその関心のある者 | 社会の動向や日常生活に役立つ知識・技術を学ぶ |
| 視覚障礙者音楽教室 | 都内の視覚障礙者 | 合奏・楽器の練習・鑑賞等 |
| 聴覚障礙者社会教養講座 | 都内の聴覚障礙者及びその関心のある者 | 社会の動向や日常生活に役立つ知識・技術を学ぶ |
| 聴覚障礙者「手話で学ぶ文盲教室」 | 都内の視覚障礙者 | 文盲に関する知識・技術を学ぶ |
| 聴覚障礙者「コミュニケーション教室」 | 都内の聴覚障礙者及びその関心のある者 | コミュニケーション方法や社会参加について学ぶ |

問い合わせ先 東京都育成庁地域支援部生涯学習課<br>TEL 03-5390-6859 FAX 03-5388-1734

# (15) ぶんまんしまぇ
(障礙者・高齢者の手作り作品代理販売事業)
