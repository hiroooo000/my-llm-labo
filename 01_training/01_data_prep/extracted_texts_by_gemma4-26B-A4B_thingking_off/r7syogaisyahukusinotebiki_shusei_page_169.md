# （４）障害者差別解消法

障害者差別解消法は、国や市区町村といった行政機関や、会社やお店などの民間事業者の、障害のある人に対する「障害を理由とする差別」をなくすための決まりごとを定めた法律です。障害の有無にかかわらず、すべての人においの尊厳を尊重しながら共に生きる社会をつくることを目的としています。障害者差別解消法では、障害のある人への「不当な差別的取扱い」が禁止され、「合理的配慮の提供」が求められています。

| 用語の趣旨（例示） | 差別解消のための支援措置 |
| :--- | :--- |
| **不当な差別的取扱い**<br>例）言葉を持ていない人の入店を拒否する。<br>・障害のあることを理由にスポーツクラブへの入会を断る。<br>正当な理由がないのに、障害を理由としてサービスの提供を拒否したり、障害のある人には不利な条件をつけたりすること。 | ・国の行政機関<br>・地方公共団体<br>・民間事業者<br>**禁止** |
| **合理的配慮の提供**<br>例）障害や事情に応じて差別をなくす。<br>・障害のある人から、「社会的な障壁」を取り除くための合理的配慮の要望表明があったとき、負担になりすぎない範囲で対応すること。<br>・病院や講演会などで、障害のある人の障壁に応じて差別をなくす。<br>・障害のある人から、「自分で書くのが大変だから代わりに行ってほしい」と申し出があった時、代わりに書いてほしい。<br>・書くに問題がある場合は、その人の意思を十分に確認しながら障害をなくす。 | ・国の行政機関<br>・地方公共団体<br>・民間事業者<br>**義務**<br><br>※民間事業者は、令和6年4月1日から義務化 |
