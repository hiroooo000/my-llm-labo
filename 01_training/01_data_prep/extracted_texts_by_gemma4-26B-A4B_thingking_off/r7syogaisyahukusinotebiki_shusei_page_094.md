# 8 障害・緊急時

## (1) 救急代理通報システム

| 身障 |
| :--- |

家庭で救急やケガなどの突発的な事象があった場合、若しくは家庭内で火災が発生し、住宅用火災警報器が作動した場合に、専用通報機器を用いて、契約している民間の警備会社を通じて消防局へ救急要請を求めると同時に、避難を受けることを迅速に伝えるための救助を求めます。

**条件**
| 対象 | 18歳以上65歳以下の者とし、次の①～③全てに該当する方 |
| :--- | :--- |
| ① | 身体障害者手帳 1・2級（聴覚障害は3級以上）をお持ちの方又は特殊疾病（スモン・閉塞性動脈硬化症等、人工透析を必要とする方）の、聴覚障害（3級以上）の方 |
| ② | 家族の者が避難者を確保できる方 |
| ③ | 自宅の鍵を予備できる方（民間警備会社に預けます。） |

※65歳以上の場合、聴覚障害2・3級は、障害福祉課へ申請ください。
※その他の65歳以上の方は、高齢福祉課にご相談ください。

| 申請手続き | 身体障害者又は特殊疾病の医師の意見書をもち、下記に申請してください。申請後、民間会社と協定を結び、後日、機器を設置します。<br>※区外から転入した方は、居住地の区役所町村で発行する住民税非課税証明書が必要です。 |
| :--- | :--- |

| 費用 | 月額330円（住民税非課税者の場合は無料） |
| :--- | :--- |
