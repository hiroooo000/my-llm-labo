# 3. 精神障害者保健福祉手帳障害等級別基準

| 障害区分 | 精神障害（診断）の程度 | 症状の程度（生活機能）の要求 |
| :--- | :--- | :--- |
| 1級 | 精神障害によるものであれば、精神疾患の症状が重度であり、他人の援助がなければ、日常生活のほとんどを他人の援助がなければ行うことが困難なもの | 1. 精神的な混乱が著しく、日常生活が困難なもの<br>2. 感情、意欲、集中力の調節が困難なもの<br>3. 思考や判断が、計画的に行うことが困難なもの<br>4. 記憶・判断力が著しく、日常生活において困難なもの<br>5. 感情や認識、自意識との関係が不安定なもの<br>6. 言語の保持やコミュニケーションの選択に困難があるもの<br>7. 社会的な関わりや、一般の社会的活動が困難なもの<br>8. 感情や認識・意識に関わり、社会的生活に著しく困難なもの<br>上記1～8のうちいくつかずつに該当するもの |
| 2級 | 精神障害によるものであれば、精神疾患の症状が中等度であり、他人の援助がなければ、日常生活のほとんどを他人の援助を受ける必要があるもの | 1. 精神的な混乱が著しく、日常生活はできないもの<br>2. 感情、意欲、集中力の調節が困難なもの<br>3. 思考や判断が、計画的に行うことが困難なもの<br>4. 記憶・判断力が著しく、日常生活において困難なもの<br>5. 感情や認識、自意識との関係が不安定なもの<br>6. 言語の保持やコミュニケーションの選択に困難があるもの<br>7. 社会的な関わりや、一般の社会的活動が困難なもの<br>8. 感情や認識・意識に関わり、社会的生活に著しく困難なもの<br>上記1～8のうちいくつかずつに該当するもの |
| 3級 | 精神障害によるものであれば、精神疾患の症状が軽度であり、他人の援助がなければ、日常生活を営むことができるもの | 1. 精神的な混乱が著しく、日常生活が困難なもの<br>2. 感情、意欲、集中力の調節が困難なもの<br>3. 思考や判断が、計画的に行うことが困難なもの<br>4. 記憶・判断力が著しく、日常生活において困難なもの<br>5. 感情や認識、自意識との関係が不安定なもの<br>6. 言語の保持やコミュニケーションの選択に困難があるもの<br>7. 社会的な関わりや、一般の社会的活動が困難なもの<br>8. 感情や認識・意識に関わり、社会的生活に著しく困難なもの<br>上記1～8のうちいくつかずつに該当するもの |
| | | ・その他の精神障害によるものにおいては、上記1～7に準ずるもの |
