# （５）重度心身障害者手当（都制度）　　　　　　　　　　　　　　　　　　知児

心身に重度の障害を有するため、常時継続な介護を必要とする方に対して支給されます。

| 対象 | 次のいずれかに該当する方 |
| :--- | :--- |
| | 介護者が常に暇せずかつ介護に特別な配慮が必要で、次の①～③のいずれかに該当する方 |
| | ① 重度の肢体すべての機能障害のため、寝違っていることが困難な方 |
| | ② 重度の認知症と重度の身体障害が重複している方 |
| | ③ 重度の知的障害でかつ著しい精神症状の方<br>（ただし、精神障害及び認知症は除く） |
| | ① 新規申請時に65歳以上のいる方 |
| | ② 施設に入所している方 |
| | ③ 病院又は診療所に継続して3か月を超えて入院している方 |
| | ④ 本人（20歳未満）は、配偶者又は養育義務者に所得が基準額を越えている方（←20ページ） |

| 必要書類 | ① 身体障害者手帳又は肢の手帳 |
| :--- | :--- |
| | ② 印鑑 |
| | ③ 申請者のマイナンバーカード又は通知カード<br>※申請者が20歳未満の場合は次のどちらか<br>・養育義務者のマイナンバーカード<br>・養育義務者の通知カードと身分証明書<br>※後日、東京都心身障害者福祉センターで診察（判定）を受けていただきます。 |

| 手当額 | 月額60,000円 |
| :--- | :--- |

# （６）特別児童扶養手当（都制度）　　　　　　　　　　　　　　　　　　知児

障害のある20歳未満の子どもを養育する方に支給されます。

| 対象 | 次のいずれかに該当する20歳未満の子どもを養育する父又は母又は養育者 |
| :--- | :--- |
| | ① 身体障害者手帳1～3級（一部4級）程度 |
| | ② 愛の手障1～3度程度 |
| | ③ 上記①と②の両度身体又は精神の障害もしくは疾病 |
| | ④ 複数の障害がある方（個々の障害の程度が上記より軽度な場合でも該当することがあります） |

| 受給できない方 | ① 申請者とその配偶者及び養育義務者に一定以上の所得のある方<br>（←205ページ） |
| :--- | :--- |
| | ② 子どもが施設に入所している方 |
| | ③ 子どもが障害を理由とする公的年金を受給している方 |
