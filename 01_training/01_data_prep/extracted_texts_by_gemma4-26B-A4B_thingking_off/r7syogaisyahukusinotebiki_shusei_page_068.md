# 6 福祉用具等の給付

## （1）補装具の支給　　　　　　　　　　　　　　　　身障児

身体障害のある方及び難聴者等が日常生活や就学、就労のために、身体機能を補完・代替する補装具を製作・修理する場合に補装具を支給します。

| 対象 | 身体障害者手्सのを受けた方、難聴者等（対象疾病 別表199ページ） |
| :--- | :--- |
| | ※次の①～③に該当する場合、対象にはなりません。 |
| | ① 障礙のある方と配偶者に、一定所得以上の方がいる場合<br>（障礙を除く） |
| | ② 介護保険の対象者<br>（介護保険の補装具用具で個別の身体状況に対応できない場合を除く） |
| | ③ 医療保険により補装具を作成した場合 |
| | ④ 労災による障礙の場合 |
| | ⑤ 補装具の製作・修理を開始している場合 |

| 補装具別 | 補装具品目 |
| :--- | :--- |
| 視覚障礙 | 視覚障礙者安つ全、眼鏡、眼鏡 |
| 聴覚障礙 | 補聴器（人工耳声信号装置の修理に限る） |
| 肢体不自由 | 義手、義足、下肢装具、腕装具、体幹装具、上肢装具、腰部保持姿勢、申請用歩行補助、電動車椅子、車椅子、歩行器、補行補助、て |
| 肢体不自由児 | 重度障礙児用座位保持装置<br>※児童のみ対象：座位保持椅子、起立保持椅子、頭部保持椅子、便携補助椅子 |
| 内障障礙 | 車椅子 |

**申請手続**　補装具の製作・修理等を開始する前に、下記窓口へ相談してください。
※補装具は、判定により支給が認められた場合に限り支給されるものであり、県立身障者補装具センターでの判定が必要な場合は、決定までに2～3か月がかることがあります。
