# (4) 特別障害者手当等（国制度）

身体又は精神に著しい重度の障害を有する方に対して支給される手当です。

| 対象 | 20歳以上で、著しく重度の障害の状態にあるため、日常生活において常に特別な介護を必要とする次のどちらかに該当する方 |
| :--- | :--- |
| | ① 重度障害が重い方 |
