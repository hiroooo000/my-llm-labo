# 5 医 療

## (1) 心身障害者医療費助成（マル障）

医療保険を使って病院や診療所等、薬剤の支給を受けた場合に、自己負担分から一部負担金を差し引いた額を助成します。

| | |
| :--- | :--- |
| **対 象** | 身障手帳1・2級（内障障を含む3級）<br>愛の手帳1・2度 |
| **助成制限** | 精障保険被保険者手帳1級<br>ただし、次れいが該当する場合は、対象外となります。<br>① 健康保険に加入の方<br>② 生活保障を受ける中の方<br>③ 都規則第5条に規定する施設に入所している方<br>（医療費が賄われている施設に入所している方）<br>④ 後高齢者手助け又は被保険者又は世帯主の所得が限度額を超えている方<br>（204ページ）<br>⑤ 重度障害者になった年齢が65歳以上である方<br>⑥ 重度障がい者になった年齢が65歳未満に達する日前日まで、マル障受給者の交付申請を行わなかった方 |
| **申請手続** | 次のものをお持ちの上、窓口で申請してください。<br>・身障手帳又は愛の手帳<br>・被保険者の「資格証明書」等<br>・申請者のマイナンバーカード又は通知カード<br>※申請者が20歳未満の場合は、次のうちどちらか<br>・被保険者の世帯主のマイナンバーカード<br>・被保険者の世帯主の通知カード又は身分証明書<br>・住民税医療費の医療費標準負担額・生活療養標準負担額をお持ちの方<br>・住民税が非課税の「受給者 食」をお持ちの方 |
| **助成の範囲** | 健康保険の自己負担分から一部負担金を差し引いた額<br>（入院時の食事標準負担額・生活療養標準負担額を除く）<br>・住民税が非課税の「受給者 食」をお持ちの方<br>健康保険の自己負担分<br>（入院時の食事標準負担額・生活療養標準負担額を除く） |
| **助成方法** | 【都と契約している医療機関】<br>受給者の番号の分かる書類を医療機関に提示すると、その場で自己負担が軽減されます。 |
