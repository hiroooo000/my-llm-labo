# 13 住宅

## (1) 住宅設備改善の助成

身体障害のある方及び難聴患者（対象者 別表 199ページ）が日常生活を容易にするために、自宅の設備改善に要する費用の一部を助成します。助成に当たっては、事前要件の相談が必要です。また、難聴患者等は医師の診断書が必要です。なお、介護保険対象者は、介護保険住宅改修を原則として優先されます。

| 利用者負担 | 原則、各種目の基準額の1割（基準額を超えて住宅設備改善をする場合、超えた額も利用者負担） |

| 対象 | 障礙児 | 年齢 | 内容 |
| :--- | :--- | :--- | :--- |
| 袖目 | | | |
| 小規模改修 | ①下肢は体幹1～3級及び補装具として車椅子の交付を受けた内部障害者<br>②聴力は体幹に障がいのある難聴患者 | 学齢期以上 65歳未満 | ・手すりの取付け<br>・段差の解消<br>・滑り防止、移動の円滑化のための床材変更<br>・浴槽等への扉の設置<br>・便器等の便器の取替え（特殊便器への取替えは上肢2以上が必要）<br>・その他の工事に付随する必要な工事 |
