# 12 税の軽減

## (1) 所得税・住民税の障害者控除

納税者自身が障害者であるとき又はその同一生計配偶者が障害者であるときは、所得金額から控除額を差し引くことができます。

**対象**
① 身体障害者手帳1～6級
② 愛の手帳1～4度
③ 精神障害者保健福祉手帳1～3級
④ 精神上の障害により判断能力を欠く常況にある方等
⑤ 精神又は身体に障害のある65歳以上の方で①から④に準ずるものとして市区町村長等に認定を受けている方

**区分**
特別障害者※（身体1・2級、知的1・2度、精神1級）
障害者（身体3～6級、知的3・4度、精神2・3級）

**控除額**
| 控除の種類 | 区分 | 所得税 | 住民税 |
| :--- | :--- | :--- | :--- |
| 障害者控除 | 障害者 | 27万円 | 26万円 |
| | 特別障害者※ | 40万円 | 30万円 |
| | | 同居特別障害者 | 75万円 | 53万円 |
