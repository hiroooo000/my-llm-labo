# (14) 東京都庁地域支援部生涯学習事業

視覚・聴覚障礙者のための講習等を行っています。費用は原則無料です。

| 講座名 | 対象 | 内容 |
| :--- | :--- | :--- |
| ① 視覚障礙者教養講座 | 都内の視覚障礙者及びその関心のあ る者 | 社会の動向や日常生活に役立つ知識・技術を学ぶ |
| 視覚障者音楽教室 | 都内の視覚障礙者 | 合奏・楽器の練習・鑑賞等 |
| ② 聴覚障礙者社会教養講座 | 都内の聴覚障礙者及びその関心のあ る者 | 社会の動向や日常生活に役立つ知識・技術を学ぶ |
| 聴覚障礙者「手話で学ぶ文章教室」 | 都内の聴覚障礙者 | 文章表現に関する知識・技術を学ぶ |
| 聴覚障礙者「コミュニケーション教室」 | 都内の聴覚障礙者及びその関心のあ る者 | コミュニケーション方法や社会参加について学ぶ |

問い合わせ先 東京都庁地域支援部生涯学習部
TEL 03-5320-6859 FAX 03-5388-1734

# (15) ぶんまんしゃして
(障礙者・高齢者の手作り作品販売事業)

内の障礙者、高齢者の方の手作り作品、障礙者施設等の商品を代理で販売するお店です。生きがいづくりや模範的な社会参加を認めるとともに、周囲の理解を深めることを目指しています。出品にあっては説明がありますので、新規の方は事前にご連絡ください。

出品できる方
① 区内在住、在勤は区内の障礙者に通っている障礙者の方
② 区内在住60歳以上の方

| 受付・問い合わせ先・販売 | 販売場所 |
| :--- | :--- |
| 文京シビックセンター 3階障礙者会館 | 障礙者施設内 文京のハート工房 (障礙者施設商品販売会)、区内イベント等 |
| 文京総合福祉センター1階 リアン文京カフェBUNBUN (文京区小日向2-16-15) | ぶんまんストア (文京総合福祉センター1階)、区内イベント等 |
