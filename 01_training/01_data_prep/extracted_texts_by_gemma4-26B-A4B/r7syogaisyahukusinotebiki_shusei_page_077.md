# (3) 小児慢性特定疾病児童日常生活用具の給付

在宅の小児慢性特定疾病児童の日常生活を容易にするため、日常生活用具の給付を行います。

| 対象 | 別表に掲げる対象者で、東京都小児慢性特定疾病医療受給者証をお持ちの方 |
| :--- | :--- |
※児童福祉法又は障害者総合支援法による用具の給付を受けることができる方は対象となりません。

### 別 表

| 種目 | 対象者 | 機能等 |
| :--- | :--- | :--- |
| 便器 | 排泄を要する者 | 小児慢性特定疾病児童が容易に使用し得るもの（手すりをつけることができる） |
| 特殊マット | 寝たきりの状態にある者 | 褥瘡の予防又は失効による潰瘍若しくは褥瘡を防止する機能を有するもの |
| 特殊寝器 | 上肢機能に障害のある者 | 足部を床に置いて温痛感出し得るもの ただし、取替えに当たり住宅改修を伴うを除く。 |
| 特殊袴 | 寝たきりの状態にある者 | 腰、臀部の訓練できる器具を付し、原則として使用者の頸部及び臀部の傾斜角度を調節できる機能を有するもの |
| 歩行支援用具 | 下肢が不自由な者 | おおむね次のような機能を有する手すり、スロープ等である。 で 小児慢性特定疾病児童の身体機能を除く |
| 入浴補助用具 | 入浴に介助を要する者 | 入浴の移動、座位の保持、浴槽への入浴を補助でき、小児慢性特定疾病児童又は介助者が容易に使用し得るもの |
| 特殊尿器 | 自力で排尿できない者 | 尿が自動的に吸引されるもので、小児慢性特定疾病児童又は介助者が容易に使用し得るもの |
| 体位変換器 | 寝たきりの状態にある者 | 介助者が小児慢性特定疾病児童の体位を変換させるのに容易に使用し得るもの |
| 車椅子 | 下肢が不自由な者 | 小児慢性特定疾病児童の身体機能を十分備えたものであって、必要な強度と安定性を有するもの |
