# (8) 児童育成手当（区制度）　　　　　　　　　　身障児

児童育成手当は条例に基づく制度で、場合に応じ、以下の2種類があります。いずれも所得制限があります。（→205ページ）

## 【障害手当 － 子どもに障害がある場合 － 】

| | |
| :--- | :--- |
| **対 象** | 次のいずれかに該当する20歳未満の子どもを養育する父母又は養育者<br>① 身体障害者手帳 1～2級度<br>② 愛の手帳 1～3度程度<br>③ 難病症又は進行性難病症 |
| **受給できない方** | ① 一定以上の所得のある方<br>② 子どもが施設入所している方 |
| **必要書類** | ① 身体障害者手帳・愛の手帳<br>② 申請者及び配偶者のマイナンバーカード又は通知カード<br>③ 預金通帳又はキャッシュカード（※申請者名義）<br>④ 申請者の身分証明書（マイナンバーカードを持参の場合は不要） |
| **手 当 金** | 子ども1人につき月額15,500円 |

## 【育成手当 － 養育する方に障害がある場合 － 】

| | |
| :--- | :--- |
| **対 象** | 次のいずれかに該当する、18歳まで（18歳の誕生後最初の3月31日まで）の子どもを養育する父母又は養育者<br>① 父母が棄児<br>② 父母が死刑<br>③ 父母に重度の障害がある<br>④ 父母は生死不明<br>⑤ 父母には1年以上虐待されている<br>⑥ 婚姻によらず生まれ、父母又は母から扶養されていない |
| **受給できない方** | ① 一定以上の所得のある方<br>② 子どもが施設入所している方 |
| **必要書類** | ① 申請者及び子どもの戸籍謄本<br>② 父又は母に障害のある方、身体障害者手帳・診断書<br>③ 申請者及び配偶者のマイナンバーカード又は通知カード<br>④ 預金通帳又はキャッシュカード（※申請者名義）<br>⑤ 申請者の身分証明書（マイナンバーカードを持参の場合は不要） |
| **手 当 金** | 子ども1人につき月額13,000円 |
