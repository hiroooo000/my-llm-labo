# 13 住 宅

## (1) 住宅設備改善の助成 身障児

身体障害のある方及び難聴患者（対象者 別表199ページ）が日常生活を容易にするために、自宅の設備改善に要する費用の一部を区が助成します。助成に当たっては、事前要件等の相談が必要です。また、難聴患者等は医師の診断書が必要です。なお、介護保険対象者は、介護保険住宅改修が原則として優先されます。

**利用者負担** 原則、各種目の基準額の1割（基準額を住宅設備改善を越える場合、越えた額も利用者負担）

| 対象 | 障礙児 | 年齢 | 内容 |
| :--- | :--- | :--- | :--- |
| 種目 | | | |
| 小規模改修 | ①下肢は体幹1～3級及び補装具として車椅子の交付を受けた内障障害者<br>②下肢は体幹に障がいのある難聴患者等 | 学齢児以上<br>65歳未満 | ・手すりの取付け<br>・段差の解消<br>・滑り防止、移動の円滑化のための床材変更<br>・引き戸等の扉の取替え<br>・浴槽便器等の便器の取替え（特殊便器への取替えは上肢2級以上）<br>・その他上記の工事に付随して必要な工事 |
| 中規模改修 | 下肢又は体幹1～2級及び補装具として車椅子の交付を受けた内障障害者 | 学齢児以上<br>65歳未満 | ・小規模改修において給付対象となる改修で、小規模改修の給付を受けてなお足りない部分についての工事<br>・小規模改修の対象とならない改修で、区が必要と認める工事 |
| 室内移動設備 | 歩行不能で上肢・体幹1級及び補装具として車椅子の交付を受けた内障障害者（65歳以上の内障障害者は、障定により補装具として車椅子が必要であるとの診断を受けた方） | 学齢児以上 | 十分な強度を持ち、天井への取付け型は、4本支柱のレールに障壁サポートを付し、身体を持ち上げ障害を安全に室内移動できる性能を有するもの |
