# (3) 利用者負担

利用者負担は、収入、所得に応じて設定されており、国・区の軽減制度があります。

## (1) 月ごとの利用者負担上限額

障害福祉サービス及び障害児支援の利用者負担は、所得に応じて区の区分が負担上限額が設定され、月には利用したサービス量にかかわらず、それ以上の負担は生じません。

| 区分 | 世帯の所得状況 | 負担上限額 |
| :--- | :--- | :--- |
| 生活保護 | 生活保護受給世帯 | |
| 低所得 | 低所得1<br>区町村民税非課税世帯のうち、本人の年収80万円以下 | 0円 |
| | 低所得2<br>区町村民税非課税世帯のうち、低所得1に該当する方を除く | |
| 一般1 | 区町村民税非課税世帯のうち所得が16万円未満の方<br>居宅で生活する障害 | 4,600円 |
| | 施設入所者の場合は28万円未満の場合 | 9,300円 |
| 一般2 | 区町村民税非課税世帯のうち、一般1に該当する方を除く | 37,200円 |

※負担の上限額は、課税世帯の場合、「一般2」のみとなります。
また、一定所得以上の世帯の方は、対象外となります。
※入所利用者（20歳以上）、グループホームの方で課税世帯の場合、「一般2」となります。

## (2) 国の軽減制度

### ① 医療型別減免
医療型施設入所者に係る療養介護を利用する方は、障害福祉サービス費の利用者負担と医療費、食事療養費を合算して上限を設定し、限度額を上回る額について減免を行います。

### ② 食事等負担減の軽減
通所施設の負担減、施設入所にかかる食費・光熱費の負担減、グループホームの家賃負担減措置です。
