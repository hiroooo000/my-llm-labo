# 17 余暇活動の支援

### (1) 心身障害者・児バスレクリエーション 身知児
障害のために、日頃行事の機会が少ない方を、年1回バス旅行等にご招待します。

| | |
| :--- | :--- |
| 対象 | 区内在住の心身障害者又は肢の障害をお持ちの方 |
| 実施方法 | 日帰りバス旅行等 |
| 申込 | 旅行先・日程・申し込み方法等詳細は、区報「ぶんきょう」、区ホームページでお知らせします。 |

問い合わせ先：障害福祉課障害者在宅サービス係
TEL 03-5803-1212 FAX 03-5803-1352

### (2) 障害者団体バス借上経費助成
区内の障害者・児及びその保護者等によって構成する団体が、バスを借り上げて事業を行った場合に、借上経費の一部を助成します。助成に当たっては、実施日の1ヵ月前までにあらかじめ申請が必要です。

問い合わせ先：障害福祉課障害者在宅サービス係
TEL 03-5803-1212 FAX 03-5803-1352

### (3) みんなのスポーツひろば 知児
「みんなのスポーツひろば」は、気持ちよくからだを動かし、みんなとの交流を楽しむ場です。

| | |
| :--- | :--- |
| 対象 | 原則として区内に在住・在勤・在学する、肢の障害をお持ちの方 |
| 内容 | チアダンスやボッチャ、ホッケーなど誰もが気軽に参加できるスポーツ<br>※身障向け、自分のバスで参加できます。 |
| 費用 | 無料（交通費等は各自負担） |
| 日時 | 5月〜翌年3月の weekend |
| 申込 | 申込方法はホームページを確認 区HP |

問い合わせ先：スポーツ振興課スポーツ振興係（シビックセンター17階北側）
TEL 03-5803-1308 FAX 03-5803-1356
