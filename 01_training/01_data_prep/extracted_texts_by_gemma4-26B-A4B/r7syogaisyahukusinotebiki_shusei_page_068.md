# 6 福祉用具等の給付

## (1) 補装具の支給 身障児

身体障害のある方及び難聴児等が日常生活や就学、就労のために、身体機能を補完・代替する補装具を製作・修理する場合に補装具を支給します。

| 対象 | 身体障害者等のを受けた方、難聴児等(対象疾病 別表199ページ) |
| :--- | :--- |
| | ※次の①～⑤に該当する場合、対象にはなりません。<br>① 障害のある方及び障児に、一定所得以上の方がいる場合 (障児を除く)<br>② 介護保険の対象者 (介護保険の補装具で個別の身体状況に対応できない場合を除く)<br>③ 医療保険により補装具を作成した場合<br>④ 労災による補装具の場合<br>⑤ 補装具の製作・修理を開始している場合 |

| 障障害 | 補 装 具 品 目 |
| :--- | :--- |
| 視覚障害 | 視覚障害者用メガネ、眼鏡、眼底 |
| 聴覚障害 | 補聴器、人工耳、人工音声信号処理装置の修理に限る |
| 肢体不自由 | 義手、義足、下肢装具、肩装具、体幹装具、上肢装具、腰用持続装具、申請用移動補装具、電動車椅子、車椅子、歩行器、歩行補助具、 |
| 肢体不自由児 | 重度障害用意思伝達装置 |
| 内部障害 | 車椅子 |

### 申請手続
補装具の製作・修理等を開始する前に、下記窓口へ相談してください。
※補装具は、判定により必要が認められた場合に限り支給されるものであり、重度心身障害者認定センターでの判定が必要な場合は、決定までに2～3か月がかかることがあります。

### 利用者負担
原則、各種の基準の1割
※基準額を超えて購入・修理等を行う場合は、超えた額を利用者が負担。
