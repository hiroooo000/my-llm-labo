# (7) JR旅客運賃の割引 [身知]

障礙者の方や介護者がJRを利用する際、運賃が割引になります。
割引をご利用の際は、身体障害者手帳又は旅客鉄道株式会社旅客運賃減額様式に第1種又は第2種の記載のあるもの)が必要となります。詳しくは、JR各駅にお問い合わせください。

| 内容・対象 | 対象 | 割引対象乗車券 | 割引率 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| 第1種障礙者とその介護者 | | 普通乗車券 | 50% | 私鉄等他鉄道会社とつながる場合を含みます。ただし、回数乗車券は、JR接続区間単発の発券となります。 |
| 第1種障礙者とその障礙者又は12歳未満の障礙者とその介護者 | | 定期乗車券 | 50% | 私鉄等他鉄道会社とつながる場合を含みます。小児定期旅客運賃については、割引を適用します。 |
| 第1種、第2種障礙者が単独で利用する場合 | | 普通乗車券 | 50% | 片道の障礙者キロが100キロを超える場合（私鉄等他鉄道会社につながる場合を含ます。） |

※障礙者と介護者がご利用の場合は、同一区間の乗車券をお願いください。
※大人の第1種障礙者が片道の普通乗車券を100キロお願いいただく場合は、自動券売機にて購入の普通乗車券をお願いください。その普通乗車券と障礙者手帳をお持ちいただければ、代理の方が障礙者割引をご利用いただくこともできます。
※障礙者がお求めいただくこともできます。（代理の方の必要書類はございません。）
