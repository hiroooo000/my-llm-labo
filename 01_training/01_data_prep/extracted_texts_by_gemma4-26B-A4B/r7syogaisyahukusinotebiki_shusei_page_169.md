# (4) 障害者差別解消法

障害者差別解消法は、国や市区町村といった行政機関や、会社やお店などの民間事業者の、障害のある人に対する「障害を理由とする差別」をなくすための決まりごとを定めた法律です。障害のあるなしにかかわらず、すべての人のおいの人権と個性を尊重し、ながらも共に生きる社会をつくることを目的としています。障害者差別解消法では、障害のある人の「不当な差別的取扱」が禁止され、「合理的配慮の提供」が求められています。

| 用法の説明（例示） | 差別解消のための支援措置 |
| :--- | :--- |
| 不当な差別的取扱<br>(例) ・言葉や態度を用いて人の入店を拒否する。<br>・障害のあることを理由にスポーツに入るのを断る。 | 禁止<br>・国の行政機関<br>・地方公共団体<br>・民間事業者 |
| 合理的配慮の提供<br>(例) ・障害のある人から、「社会的な障壁」を取り除くための合理的配慮の要望が明らかとなったら、負担になりすぎない範囲で対応すること。<br>・病院や公共会館などで、障害のある人の入店を禁止にする代わりに「自分に書ける言葉で書いてほしい」と申出があった時、代わりについてほしいと申し出た場合、その人の意思を十分に確認しながら障害を軽減する。 | 義務<br>・国の行政機関<br>・地方公共団体<br>・民間事業者<br><br>※民間事業所は、令和6年4月1日から義務化 |
