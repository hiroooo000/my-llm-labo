# (5) 障害者虐待の防止

障害者虐待は、どこの家庭や施設・職場でも起こり得る問題です。また、虐待を受ける人、してしまう人が虐待だと認識せず、自分から助けを求められない場合があります。

虐待を防ぐためには、住民一人ひとりが正しい知識を身につけ、小さな兆候を見逃さず、早期発見に努めることが大切です。

**障害者虐待の防止**
障害者虐待の防止は、虐待は以下の3種類に分けています。
① 養護者による虐待（家族・親族等）
② 障害福祉施設従事者等による虐待（施設職員・ホームヘルパー等）
③ 使用者による虐待（仕事上の事業主等）

| 障害者虐待の形態 | |
| :--- | :--- |
| 身体的虐待 | 暴行や叩き、体に傷や痛みを強いる。 |
| 性的虐待 | 無理やり性的なことを、させる。 |
| 心理的虐待 | 言葉や態度で、精神的苦痛を与える。 |
| 放棄・放置（ネグレクト） | 世話や介助をせず、心身を衰弱させる。 |
| 経済的虐待 | 本人の同意なく、財産や年金、賃金などを使う。 |

障害者虐待に気づいた人には、市区町村の相談窓口への通報義務があります。通報や届出をした人の個人情報は、守秘義務により守られます。また、匿名による通報も受け付けています。虐待の兆候に気づいたら、ご連絡・ご相談ください。
