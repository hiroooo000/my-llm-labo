# 16 仕事

## (1) 文京区障害者就労支援センター 身寄り困難

障害者の就労に関する相談窓口として、障害者や家族・企業・関係機関からの相談を受け、障害者施設やハローワークと連携しながら、各種就労に向けての支援を行います。

| 支援 | 支援の内容 |
| :--- | :--- |
| 就労準備支援 | 適性や能力に応じた就労への支援（キャリアカウンセリング、職業訓練の支援、就職活動支援、履歴書・職務経歴書の作成支援、模擬面接や面接同行等） |
| 職場実習支援 | 就業を目指す方へ、職場体験を通じた職場環境をコーディネート、実習先での事業主等に対する理解を深め、職場環境の調整を行う等の支援 |
| 職場定着支援 | 安心して働き続けられるように、就職後の継続したフォロー、定期的な面談、職場での相談、職場訪問による環境の調整等 |
| 生活支援 | 働くことを継続するために必要な、生活リズムの調整、健康管理、余暇など職業生活に関する相談・助言 |

※その他就労について必要な支援については、お気軽にご相談ください。
※就労相談は、登録制です。

| 費用 | 無料 |
| :--- | :--- |
| 受付 | 月〜金曜 午前9時〜午後5時 (祝日及び年末年始を除く) |
| 相談・問い合わせ | 文京区障害者就労支援センター（文京区民第1階）<br>〒113-0033 文京区本駒込4－15－14<br>TEL 03-5805-1600 FAX 03-5805-1601 HP |

## (2) 波田窪 職業安定所（ハローワーク波田窪） 身寄り困難

就労することを希望する障害の方への仕事の情報提供、紹介、就職活動の相談、必要な技能習得のための職業訓練への入校相談を行っています。
また、障害者の募集、雇用する企業への助言、雇用後の雇用管理やキャリア支援に関する助言、雇用における合理的配慮等に関する助言

| 受付 | 月〜金曜 午前8時30分〜午後5時半 9窓口 |
※手話通訳付き相談の日程については、お問い合わせください。
