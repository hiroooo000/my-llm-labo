# (9) 医療的ケア児在宅レスパイト事業

在宅で生活する医療的ケア児の健康保持、介護する同居の保護者等の介護負担の軽減、及び就労・就職活動の支援を図るため、看護師又は看護助手を派遣し、一定時間、医療的ケア等を行います。

**対象** 以下のいずれにも該当する方
① 市区町村の住民に登録を有し、かつ居住の実態がある方
② 18歳までの方（18歳の誕生後初の3月31日まで）
③ 別表に規定する医療的ケアのうち、1つ以上を受けている方
④ 保護者等による在宅介護を受けて生活している方
⑤ 訪問看護により医療的ケアを受けている方

**別表**
医療的ケア（以下のうち、1つ以上のケアを受けていること。）

| | |
| :--- | :--- |
| ① 人工呼吸管理<br>(*当日機械の気道確保を要する方) | ② 中心静脈栄養 IVH |
| ③ 気管内吸引、気管切開 | ④ 経鼻栄養 (経鼻・胃ろう含む) |
| ⑤ 鼻腔ドレナージ | ⑥ 胃ろう・胃管栄養 |
| ⑦ 腹膜透析 (腹膜透析含む) | ⑧ 定期導尿 (3回/日以上)<br>(*人工膀胱を含む) |

**派遣時間** 1回当たり2時間から4時までの30分単位
**上限** 年度 (4月〜翌3月) の間に288時間を上限
**費用** 自己負担は、市区町村税の課税状況に基づき算定します。
**問い合わせ先** 障害福祉課障害者支援係
