# (2) 成年後見制度利用助成事業

成年後見制度を利用する方のうち、後見人、保佐人又は補助人の報酬に係る費用等を負担することが困難で、かつ一定の要件を満たす方に対して、報酬費用等を助成します。
※なお、中立費用助成については社会福祉協議会（権利擁護センター）が実施しています。（下記参照）

# (3) 権利擁護センター「あんしんサポート文京」

権利擁護センター「あんしんサポート文京」では、障害者が安心して地域社会での生活が続けられるように各種の支援を行っています。

| 支援の種類 | 支援の内容 |
| :--- | :--- |
| 福祉サービス利用助成事業 (地域福祉擁護事業) | 知的障害者・精神障害者等が判断力が十分でない方への福祉サービス利用に関する援助や日常的金銭管理、書書類等の支援 (1時間1,000円〜。なお、生活保障等は別紙) |
| 財産管理等支援サービス | 身体障害者等が財産の管理が困難な方への日常的金銭管理、書書類等の支援 (1時間1,000円〜。なお、生活保障等は別紙) |
| 福祉相談等 | 福祉サービス利用に関するトラブルや疑問、また、障害者の日常生活上の法律に関する事柄についての弁護士相談 (個別相談・予約が必要) |
| 成年後見制度利用相談 | 成年後見制度・財産管理の利用についての弁護士・司法書士相談 (個別相談・予約が必要) |
| 成年後見制度の利用支援 | 成年後見制度の利用に関する相談、支援。また、成年後見制度に関する学習会を開催 |
| 成年後見制度中立費用助成 | 低所得の方の由で成年後見制度（法定後見）の中立費用助成を行い、なお権利擁護成年後見人の報酬助成は、福祉政策が実施しています。（上記参照） |
