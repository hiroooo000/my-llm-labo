

# (8) 児童育成手当（区制度）

児童育成手当は条例に基づく区の制度で、場合に応じ、以下の2種類があります。いずれも所得制限があります。（→205ページ）

## 【障害手当】 — 子どもに障害がある場合 —

| 項目 | 内容 |
| :--- | :--- |
| **対　象** | 次のいずれかに該当する20歳未満の子どもを養育する父母又は養育者<br>① 身体障害者手帳1・2級程度<br>② 愛の手帳1〜3度程度<br>③ 脳性麻痺又は進行性筋萎縮症 |
| **受給できない方** | ① 一定以上の所得のある方<br>② 子どもが施設入所している方<br><br>※保護者が区内に住んでいない場合は、心身障害者等福祉手当に該当することがあります。 |
| **必要書類** | ① 身体障害者手帳・愛の手帳<br>② 申請者及び配偶者のマイナンバーカード又は通知カード<br>③ 預金通帳又はキャッシュカード（※申請者名義）<br>④ 申請者の身分証明書（マイナンバーカードを持参の場合は不要） |
| **手当額** | 子ども1人につき月額15,500円 |

## 【育成手当】 — 養育する方に障害がある場合 —

| 項目 | 内容 |
| :--- | :--- |
| **対　象** | 次のいずれかに該当する、18歳まで（18歳の誕生日後の最初の3月31日まで）の子どもを養育する父母又は養育者<br>① 父母が離婚<br>② 父又は母が死亡<br>③ 父又は母に重度の障害がある<br>④ 父又は母が生死不明<br>⑤ 父又は母に1年以上遺棄されている<br>⑥ 父又は母が配偶者からのDVにより、裁判所から保護命令を受けている<br>⑦ 父又は母が1年以上拘禁されている。<br>⑧ 婚姻によらないで生まれ、父又は母から扶養されていない |
| **受給できない方** | ① 一定以上の所得のある方<br>② 子どもが施設入所している方 |
| **必要書類** | ① 申請者及び子どもの戸籍謄本<br>② 父又は母に障害のある方は、身体障害者手帳・診断書<br>③ 申請者及び配偶者のマイナンバーカード又は通知カード<br>④ 預金通帳又はキャッシュカード（※申請者名義）<br>⑤ 申請者の身分証明書（マイナンバーカードを持参の場合は不要） |
| **手当額** | 子ども1人につき月額13,500円 |
